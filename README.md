# Constellation Quiz

Веб-приложение, которое помогает изучать созвездия в интерактивной форме.  
Пользователи могут просматривать карточки созвездий, добавлять новые, редактировать существующие и проверять свои знания в режиме викторины.

### Основной функционал

| Функция | Описание |
|---------|----------|
| **Карточки созвездий** | Просмотр списка созвездий с изображениями, названиями и описаниями |
| **Режим викторины** | Интерактивная игра с угадыванием созвездий |
| **Добавление созвездий** | Возможность расширять коллекцию через форму с валидацией |
| **Редактирование** | Изменение информации о существующих созвездиях |
| **Удаление** | Удаление созвездий с подтверждением |
| **Статистика** | Подсчёт правильных ответов и процент точности в викторине |

## Технологии

- **Python** 3.13+
- **Django** 6.0.3
- **HTML5 / CSS3** (адаптивный дизайн)
- **Git** (контроль версий)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/SixroomSandwich/Constellation_quiz.git
cd Constellation_quiz
```

### 2. Создание и активация виртуального окружения

```bash
python -m venv venv
source venv/bin/activate      # Для Linux/Mac
venv\Scripts\activate       # Для Windows
```

### 3. Установка зависимостей
```bash
pip install django
```

### 4. Применение миграций
```bash
python manage.py migrate
```

### 5. Создание суперпользователя
```bash
python manage.py createsuperuser
```

### 6. Запуск сервера
```bash
python manage.py runserver
```

### 7. Открыть в браузере
Перейдите по адресу: https://127.0.0.1:8000

## Как пользоваться

### Просмотр созвездий
- Перейдите в раздел **"Все созвезия"**
- Нажмите **"Подробнее"** для просмотра полной информации

### Добавление нового созвездия
- Нажмите **"Добавить созвездие"** на главной странице
- Заполните форму (названия на русском и латинском, описание, ссылка на изображение)
- После сохранения созвездие появится в общем списке

### Редактирование созвездия
- Перейдите в раздел **"Все созвезия"**
- Нажмите **"Подробнее"** для просмотра полной информации
- Нажмите **"Редактировать"**
- Измените нужную информацию
- Нажмите **"Сохранить"**

### Удаление созвездия
- Перейдите в раздел **"Все созвезия"**
- Нажмите **"Подробнее"** для просмотра полной информации
- Нажмите **"Удалить"**
- Подтвердите удаление

### Викторина
- Нажмите **"Викторина"** на главной странице
- Угадайте созвездия по изображению, выбрав один из вариантов
- После ответа на все вопросы вы ивудите итоговый результат
- Кнопка **"Сбросить прогресс"** позволяет начать игру заново
