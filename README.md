# Constellation Quiz

Интерактивное веб-приложение для изучения созвездий.

## Что умеет приложение?

- **Карточки созвездий** - просмотр и заучивание созвездий
- **Режим викторины** - проверь свои знания в игровое форме
- **Добавление созвездий** - расширяй коллекцию самостоятельно
- **Редактирование** - исправляй или дополняй информацию
